# esp32-lcdi2c
## Подключение
`lcd scl ---> 22 lcd sda ---> 21 vcc ---> 5v GND ---> GND `

Важно:\
При подключении к 3в дисплей не заработает.
Ессли у вас esp32 на своей плате пины могут несовпадать
